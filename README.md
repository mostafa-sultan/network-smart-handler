# network-smart-handler

A smart network handler package for Web and React Native that provides intelligent network detection, automatic request queuing, retry logic, and user-friendly notifications.

## Installation


```sh
npm install network-smart-handler
```


## Usage


```js
import { multiply } from 'network-smart-handler';

// ...

const result = await multiply(3, 7);
```
